# App

CLI to support the hybrid app development.

## Installation

Run `npm install @scriptpilot/app --save-dev`

## Usage

Run `npx app <command>`

## Commands

### hello

Logs `Hello World` in the terminal.

## Package Development

### Testing

Run commands with `node cli <command>`

### Publishing

To publish a new version to NPM:

1. Bump version
2. Run `npm publish`
